# 数据类型
组合数据类型: 包括 `集合`, `有序(列表和元组)`, `字典` 三种类型

# 集合类型
* 与数学中集合概念的要求一样, 具有**无序性**和元素**唯一性**
* 集合类型要求集合中的元素**不可变**, 一旦元素被放入集合中, 这个元素就不能修改. 不可变性是为了避免违背唯一性. 
* 集合定义用 `{...}`, 用逗号分隔. 也可以用 `set()` 申明.
* 空集合必须用 `set()` 申明, 空的大括号用来声明一个空字典.
* 集合的元素顺序常常和申明时的顺序不同, 保存时的顺序是无法使用的.
```
S1 = {'python', 123, ('xuzengchao', 123)}
S2 = set('pypy123')
S3 = {'python', 123, 'python', 123}
```

集合操作符有 6 个:
* `S1 | S2`             新集合, 并集
* `S1 - S2`             新集合, 差集
* `S1 & S2`             新集合, 交集
* `S1 ^ S2`             新集合, 两个集合中的 `非相同` 元素; 可用于`补集`.
* `S1 <= S2`            True/False, 判断包含关系
* `S1 < S2`
* `S1 >= S2`
* `S1 > S2`

4 个增强操作:
* `S1 |= S2`            更新集合 S1
* `S1 -= S2`
* `S1 &= S2`
* `S1 ^= S2`

集合自带的方法:
* `S.add(x)`      如果 x 不在 S 中, 添加
* `S.discad(x)`   如果 x 不在 S 中, `不报错`
* `S.remove(x)`   如果 x 在 S 中, 产生 `KeyError` 异常
* `S.clear()`     清空集合
* `S.pop()`       随机返回 S 中的一个元素, 并从 S 中移除该元素. S 为空则 `KeyError`
* `S.copy()`      返回副本
* `len(S)`
* `set(x)`        将变量 x 转为集合类型
* `x in S`
* `x not in S`

下面这段程序用 `while` 遍历集合中的元素, 利用了 `pop()` 方法得到唯一顺序操作, 利用异常控制终止遍历.
```
A = {'A', 'B', 123, 'xuzengchao', 'limeiren'}
try:
    while True:
        print(A.pop(), end=", ")
except :
    pass
```

集合类型的应用:
* 判断包含关系, 如 `for...in...` 的遍历.
* 数据去重, 这个比列表类型遍历要简洁有效得多. `list(set(S))` 可以直接对 list 类型去重. 

# 序列类型
* 元素之间存在顺序关系, 所以元素内容可以相同.
* 元素的类型可以不同, 这和 R 的向量不同.
* 序列是基类类型, 不直接使用. 使用衍生出的类型, 如字符串,元组,列表类型.
* 元组用于元素不改变的情形(元素固定搭配), 可以保护数据.
* 列表更灵活, 最常用. 
* 表达一组数据, 能够按需要进行遍历.
* <emph style='border-bottom:1px solid red'>如果不希望数据被程序改变, 可以将其转为 tuple 进行保护</emph>, 例如:
    * `week = ['Mon', 'Sun']`
    * `week = tuple(week)`
* <emph style='border-bottom:1px solid red'>多人共同编写程序时, 可以根据需要要求使用 `tuple` 传递数据</emph>.
* 元组操作与序列操作基本相同.
* 列表操作更加丰富.

序列操通用操作符(对衍生类型可直接使用):
* `x in s`                返回 True/False
* `x not in s` 
* `s + t`                 连接两个序列(集合类型就没有这种操作, 只能用 s|t)
* `s*n` 或 `n*s`          将 s 复制 n 次(比 R 中的 rep 好用啊)
* `s[i]`                  第 i 个元素, i 是序列的序号
* `s[i:j]` 或 `s[i:j:k]`  第i到第j以k为步长的子列
序列的通用函数和方法:
* `len(s)`
* `min(s)`, `max(s)`   要求s中的元素可比较
* `sorted(s)`          排序
* `s.index(x)`         
* `s.index(x, i, j)`   返回从 i 到 j 位置中第一次出现 x 的位置. 没有出现则 ValueError
* `s.count(x)`         计数

# 元组类型
* 元组是序列类型, `一旦创建就不能被修改`.
* 用圆括号和逗号定义, 其中圆括号可以省略.
* 可用 tuple 转换.
* 元组中元素类型可以不同.
* 继承了序列类型的全部操作.
* 因为元组类型创建后不能修改, 所以 `没有` 针对元组类型的特殊方法.

# 列表类型
* 列表是序列类型, 创建后可以随意修改.
* 使用方括号和逗号定义.
* 可用 `list` 转换.
* 方括号和 `list` 真正创建一个列表, <emph style='border-bottom:1px solid red'>列表的等号传递 `lsNew = lsOld` 没有申明新的列表</emph>.

函数和方法:
* `ls[i] = x`                        替换元素
* `ls[i:j:k] = lt`                   替换子列
* `del ls[i]`                        删除元素
* `del ls[i:j:k]`                    删除
* `ls += lt`                         更新 ls(这个比 ls = ls+lt 要快吧)
* `ls *= n`                          更新 ls
* `ls.append(x)` 
* `ls.clear()`
* `ls.copy()`
* `ls.insert(i, x)`
* `ls.pop(i)`                       返回第i个元素并从ls中删除
* `ls.remove(x)`                    将出现的 `第一个` 删除
* `ls.reverse()`                    <emph style='border-bottom:1px solid red'>自动更新, 不需赋值</emph>
* `ls = ['cat', 'dog', 'tiger', 1024]`
* `ls[1:2] = [1, 2, 3, 4]`          <emph style='border-bottom:1px solid red'>注意这种扩长的用法</emph>
* `del ls[::3]`
* `ls*2`

# 字典类型
* 键值对: 键(索引)和值(数据)的对应, 键是人为定义的数据索引的扩展.
* 字典是键值对的 `集合`, 键值对之间无序.
* 字典用大括号和 `dict()` 创建, 键值对用冒号表示.
* 空的大括号是用来生成空字典, 不用来生成空集合.

字典类型操作函数和方法:
* `del d[k]`               k为键
* `k in d`                 k为键[这个重要, 字典面向使用者的是键不是值]
* `len(d)`
* `d.keys()`               返回所有键信息
* `d.values()`             返回所有值信息
* `d.items()`              返回所有键值对信息
* `d.get(k, <default>)`    如果键k不存在, 返回 `<default>` 值
* `d.pop(k, <default>)`    返回并 `删除`
* `d.popitem()`            随机取出一个键值对并 `删除`, 返回 tuple 类型
* `d.clear()`

* 上面的函数和方法中, 使用者总是通过键实现对字典的操作
* 字典类型通过赋值方式增加和修改键值对.
* 字典用在表达映射的场景, 用键值对表明一一对应.
* 字典的遍历是通过键形成的: `for k in d`
